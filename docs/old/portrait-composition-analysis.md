# 인물 촬영: 구도, 포즈, 표정, 시선 - 완전 분석

Portrait Photography의 4대 핵심 요소와 충돌 분석

---

## 📐 **1. 구도 (Framing/Composition)**

### **정의**: 인물이 화면에서 차지하는 범위

| 구도 | 영문명 | 범위 | 용도 | 특징 |
|------|--------|------|------|------|
| **익스트림 클로즈업** | Extreme Close-Up (ECU) | 얼굴 일부 (눈만, 입만) | 디테일 강조, 영화적 | 감정 극대화 |
| **클로즈업** | Close-Up (CU) | 얼굴 전체 (턱~이마) | 표정 중심, 뷰티 | 피부/표정 집중 |
| **바스트샷** | Bust Shot | 가슴 상단까지 | 인물 포트레이트 표준 | 얼굴+어깨 라인 |
| **웨이스트샷** | Waist Shot | 허리까지 | 상반신+손 제스처 | 손 표현 가능 |
| **하프샷** | Half Shot | 무릎까지 | 앉은 포즈 | 다리 일부 포함 |
| **스리쿼터샷** | 3/4 Shot | 무릎 바로 위까지 | 서있는 포즈 | 거의 전신 |
| **풀샷** | Full Shot | 머리~발끝 전체 | 패션, 전신 포즈 | 전체 실루엣 |
| **롱샷** | Long Shot | 인물+배경 많이 | 환경 포트레이트 | 맥락/공간감 |

---

## 💃 **2. 포즈 (Pose)**

### **정의**: 신체의 자세와 위치

### **A. 기본 스탠딩 포즈**

| 포즈 | 설명 | 신체 특징 | 용도 |
|------|------|-----------|------|
| **스트레이트** | Straight/Frontal | 정면, 양발 평행, 양어깨 수평 | 공식적, 증명사진 |
| **컨트라포스토** | Contrapposto | 한쪽 다리에 체중, 어깨 기울어짐 | 자연스러움, 우아함 |
| **S-커브** | S-Curve | 몸통이 S자 곡선 | 여성스러움, 패션 |
| **3/4 턴** | 3/4 Turn | 몸은 비스듬, 얼굴은 카메라 | 슬림해 보임 |

### **B. 손 포즈**

| 포즈 | 설명 | 손 위치 | 분위기 |
|------|------|---------|--------|
| **자연스럽게** | Natural/Relaxed | 몸 옆에 편안히 | 일상적 |
| **에디토리얼 핸즈** | Editorial Hands | 얼굴/머리/목 근처 터치 | 패션, 세련됨 |
| **포켓 핸즈** | Hands in Pockets | 주머니에 손 | 캐주얼, 쿨함 |
| **크로스 암즈** | Crossed Arms | 팔짱 | 자신감, 때로 방어적 |
| **프레이밍** | Framing Face | 양손으로 얼굴 프레임 | 예술적, 뷰티 |
| **헤어 터치** | Hair Touch | 머리카락 만지기 | 여성스러움 |

### **C. 앉은 포즈**

| 포즈 | 설명 | 특징 |
|------|------|------|
| **리클라인** | Reclining | 기대거나 누움 | 편안함, 관능적 |
| **시팅** | Sitting | 의자에 앉음 | 안정적, 인터뷰 느낌 |
| **페치** | Perching | 가볍게 걸쳐앉음 | 활동적 |

## 😊 **3. 표정 (Expression)**

### **정의**: 얼굴에 드러나는 감정

| 표정 | 영문명 | 얼굴 특징 | 용도 | 감정 |
|------|--------|-----------|------|------|
| **자연스러운 미소** | Natural Smile | 입꼬리 살짝, 눈 웃음 | 친근함 | 행복, 편안함 |
| **활짝 웃음** | Bright Smile | 이 보임, 눈 초승달 | 즐거움 표현 | 기쁨, 에너지 |
| **은은한 미소** | Subtle Smile | 입꼬리만 살짝 | 우아함 | 만족, 차분함 |
| **중립적** | Neutral | 무표정, 이완 | 패션, 에디토리얼 | 쿨, 프로페셔널 |
| **시리어스** | Serious | 입 다물고 긴장 | 극적, 강렬 | 집중, 진지함 |
| **사색적** | Pensive/Thoughtful | 먼 곳 응시, 약간 슬픔 | 예술적 | 명상, 그리움 |
| **신비로운** | Mysterious | 애매한 표정, 모나리자 스마일 | 흥미 유발 | 수수께끼 |
| **강렬한** | Intense | 눈썹 찌푸림, 눈빛 강함 | 드라마틱 | 열정, 분노 |
| **장난스러운** | Playful | 윙크, 혀 내밀기 | 경쾌함 | 유머, 귀여움 |
| **관능적** | Sensual | 반쯤 뜬 눈, 입술 살짝 벌림 | 섹시함 | 매력, 유혹 |

---

## 👁️ **4. 시선 (Gaze/Eye Direction)**

### **정의**: 눈이 향하는 방향

| 시선 | 영문명 | 방향 | 효과 | 느낌 |
|------|--------|------|------|------|
| **카메라 직시** | Direct Eye Contact | 정면, 렌즈 응시 | 강한 연결감 | 자신감, 소통 |
| **카메라 밖** | Off-Camera | 옆/위/아래 | 자연스러움 | 사색, 편안함 |
| **위 응시** | Looking Up | 위쪽 | 희망적, 꿈꾸는 | 순수, 동경 |
| **아래 응시** | Looking Down | 아래쪽 | 내성적, 수줍음 | 겸손, 슬픔 |
| **옆 응시** | Side Gaze | 왼쪽/오른쪽 | 동적, 스토리텔링 | 호기심, 기다림 |
| **어깨 너머** | Over the Shoulder | 뒤돌아 보는 | 초대감, 신비로움 | 되돌아봄 |
| **눈 감음** | Eyes Closed | 감긴 눈 | 내면 집중 | 평화, 명상 |
| **반쯤 뜬 눈** | Half-Closed Eyes | 반쯤 감김 | 관능적, 피곤 | 나른함, 섹시 |

---

## ❌ **충돌 분석 1: 구도 ↔ 포즈**

### **물리적 불가능 (Error)**

#### **1. Extreme Close-Up + 전신 포즈**
```
ECU (얼굴 일부만) + Full Body Contrapposto
→ ❌ 얼굴만 찍는데 전신 포즈 의미 없음
```

#### **2. Close-Up + 하체 포즈**
```
CU (얼굴만) + S-Curve (전신)
→ ❌ S자 곡선은 전신이 보여야 효과 있음
```

#### **3. Bust Shot + Hands in Pockets**
```
Bust Shot (가슴까지) + Pocket Hands
→ ❌ 주머니(하체)가 화면에 안 잡힘
```

#### **4. Close-Up + Crossed Arms**
```
CU (얼굴만) + Crossed Arms
→ ❌ 팔짱이 화면에 안 보임
```

### **비효율적 (Warning)**

#### **5. Full Shot + Editorial Hands (얼굴 터치)**
```
Full Shot (전신) + Hands on Face
→ ⚠️ 전신 구도에서 손-얼굴 디테일 안 보임
→ Bust Shot이나 Waist Shot이 더 효과적
```

#### **6. Long Shot + Hair Touch**
```
Long Shot + Hair Touch
→ ⚠️ 너무 멀어서 머리카락 터치 디테일 없음
```

---

## ❌ **충돌 분석 2: 구도 ↔ 표정**

### **비효율적 (Warning)**

#### **1. Full Shot + 디테일 표정**
```
Full Shot + Subtle Smile (은은한 미소)
→ ⚠️ 전신 샷에서 은은한 미소는 거의 안 보임
→ Close-Up이나 Bust Shot 권장
```

#### **2. Long Shot + 표정 중심**
```
Long Shot + Intense Expression
→ ⚠️ 멀어서 표정 디테일 손실
→ 표정이 중요하면 Close-Up 필수
```

### **허용 가능**

#### **3. ECU + 어떤 표정도 가능**
```
ECU + Any Expression
→ ✅ 극단적 클로즈업은 모든 표정 강조
```

---

## ❌ **충돌 분석 3: 구도 ↔ 시선**

### **물리적 제약 (Error)**

#### **1. ECU (눈만) + Eyes Closed**
```
Extreme Close-Up (눈 중심) + Eyes Closed
→ ❌ 눈만 찍는데 감으면 내용 없음
→ 의도적 예술 표현 아니면 피해야
```

### **비효율적 (Warning)**

#### **2. Long Shot + Eye Contact**
```
Long Shot + Direct Eye Contact
→ ⚠️ 너무 멀어서 눈빛 전달 약함
→ Eye contact는 Close-Up에서 효과적
```

---

## ❌ **충돌 분석 4: 포즈 ↔ 표정**

### **개념적 모순 (Error)**

#### **1. Crossed Arms + Bright Smile**
```
Crossed Arms (방어적) + Bright Smile (환한 웃음)
→ ❌ 신체 언어 모순
→ 팔짱 = 닫힘, 웃음 = 열림
→ 혼란스러운 메시지
```

#### **2. Relaxed Pose + Intense Expression**
```
Natural Relaxed + Intense/Angry Face
→ ❌ 몸은 편안한데 얼굴은 긴장
→ 일관성 없음
```

### **비자연스러움 (Warning)**

#### **3. Hands on Face + Neutral Expression**
```
Editorial Hands (얼굴 터치) + Neutral/Blank
→ ⚠️ 손은 우아한데 얼굴 무표정
→ 가능하지만 어색함
→ 은은한 미소나 사색적 표정이 더 자연스러움
```

#### **4. Contrapposto + Serious**
```
Contrapposto (우아함) + Very Serious
→ ⚠️ 우아한 자세 + 과도하게 심각한 표정
→ 균형 필요
```

---

## ❌ **충돌 분석 5: 포즈 ↔ 시선**

### **개념적 모순 (Error)**

#### **1. Over-the-Shoulder + Direct Eye Contact**
```
Over Shoulder Pose (뒤돌아봄) + Looking Straight
→ ❌ 뒤돌아보는 포즈인데 정면 응시?
→ 물리적으로 어색함
→ Side Gaze나 Looking Back이 자연스러움
```

### **비자연스러움 (Warning)**

#### **2. Crossed Arms + Looking Down**
```
Crossed Arms (자신감) + Looking Down (수줍음)
→ ⚠️ 신체는 강한데 시선은 약함
→ 의도적 대비 아니면 어색
```

#### **3. Hair Touch + Eyes Closed**
```
Hair Touch + Eyes Closed
→ ⚠️ 머리 만지는데 눈 감음
→ 명상적 느낌 의도 아니면 부자연스러움
```

---

## ❌ **충돌 분석 6: 표정 ↔ 시선**

### **개념적 모순 (Error)**

#### **1. Bright Smile + Looking Down**
```
Bright Smile (활짝 웃음) + Looking Down
→ ❌ 환하게 웃는데 고개 숙임?
→ 신체 언어 모순
→ 웃으면 보통 Up or Direct를 봄
```

#### **2. Playful Expression + Eyes Closed**
```
Playful (장난스러움) + Eyes Closed
→ ❌ 장난치려면 눈 떠야 함
→ 윙크는 가능, 완전히 감으면 이상함
```

### **감정 불일치 (Warning)**

#### **3. Sad/Pensive + Direct Eye Contact**
```
Pensive (사색) + Direct Eye Contact
→ ⚠️ 슬픈데 직시?
→ 가능하지만 비자연스러움
→ 사색은 보통 Off-Camera나 Looking Away
```

#### **4. Sensual + Looking Up**
```
Sensual (관능적) + Looking Up
→ ⚠️ 섹시한 표정인데 위 응시?
→ Looking Up은 순수/희망적 느낌
→ 충돌 (Sensual은 보통 Half-Closed나 Direct)
```

#### **5. Mysterious + Bright Smile**
```
Mysterious + Bright Smile
→ ⚠️ 신비로운데 환하게 웃음?
→ 신비로움은 Subtle Smile이나 Neutral
```

## 🎯 **권장 조합 예시**

### **1. 클래식 뷰티 포트레이트**
```
구도: Close-Up
포즈: Editorial Hands (얼굴 터치)
표정: Subtle Smile
시선: Direct Eye Contact

→ 세련되고 우아한 뷰티 샷
```

### **2. 패션 에디토리얼**
```
구도: 3/4 Shot
포즈: S-Curve
표정: Neutral
시선: Off-Camera (옆 응시)

→ 쿨하고 프로페셔널
```

### **3. 자연스러운 포트레이트**
```
구도: Waist Shot
포즈: Contrapposto
표정: Natural Smile
시선: Direct or Slight Off-Camera

→ 친근하고 편안함
```

### **4. 드라마틱 샷**
```
구도: Bust Shot
포즈: 3/4 Turn
표정: Intense/Serious
시선: Direct Eye Contact

→ 강렬하고 집중적
```

### **5. 사색적 분위기**
```
구도: Waist Shot
포즈: Natural/Relaxed
표정: Pensive
시선: Looking Away (Off-Camera)

→ 내면적, 예술적
```

### **6. 활기찬 샷**
```
구도: Half Shot
포즈: 움직임 있는 자세
표정: Bright Smile
시선: Direct Eye Contact

→ 에너지 넘침
```

## ❌ **피해야 할 조합 예시**

### **1. 물리적 불가능**
```
❌ Close-Up + Hands in Pockets
   → 주머니가 화면에 안 보임

❌ ECU + Contrapposto
   → 전신 자세가 의미 없음

❌ Over-Shoulder Pose + Direct Eye Contact
   → 뒤돌아보는데 정면 응시 어색
```

### **2. 개념적 모순**
```
❌ Crossed Arms + Bright Smile
   → 닫힌 자세 + 열린 표정

❌ Bright Smile + Looking Down
   → 활짝 웃는데 고개 숙임

❌ Playful + Eyes Closed
   → 장난스러운데 눈 감음
```

### **3. 비효율적**
```
⚠️ Full Shot + Subtle Smile
   → 전신 구도에서 미세한 미소 안 보임

⚠️ Long Shot + Eye Contact
   → 너무 멀어서 눈빛 전달 약함

⚠️ Editorial Hands + Neutral Expression
   → 우아한 손 + 무표정 (어색)
```
